## README-,  age-restriction-checker.py

This is a beginner project that checks, validates and enforces age restriction on user inputs of a variety of items.

This project is built for learning purposes only; focuses on clean, structured logic and controlled program flow, rather than production-level features.

## Learning areas in focus

- Input gating
- User input handling
- Logic statements
- Item validation
- Conditional logic
- Flow control
- System design
